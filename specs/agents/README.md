# Agents Module

AI agents that can be added to a startup's team as employees. Each agent has a pre-configured role, description, tools, and resume. When added to a workspace, an agent gets its own isolated space — its own tasks, memory, thread, and calendar — while sharing the same task management system as the rest of the team.

Agents work autonomously within their scope. They see only what's assigned to them. They use the Inbox to communicate with the founder when they need human input.
