# Inbox Module

The Inbox is a human-in-the-loop communication channel between AI agents and workspace members. When an agent needs approval, has a blocker, wants to send a report, or needs any human input — it pauses execution and sends a message to the Inbox.

The Inbox is not a separate messaging system. Each inbox item is linked to the conversation thread where the agent was working. The user opens the item, sees the full thread context, replies, and the agent resumes.

Think of it as: agents email you when they need something, and your reply unblocks them.
