# Manager Module

The AI Manager is the founder's right hand. It is auto-assigned to every startup workspace and helps the founder manage their startup and team. It has full context on the workspace — startup details, team members, tasks, documents, financials — and can take actions on behalf of the founder.

The Manager is the primary interface between the founder and the workspace. Instead of navigating forms and dashboards, the founder can just talk to the Manager and get things done.
